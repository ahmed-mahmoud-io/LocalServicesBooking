🛠️ Local Services Booking System

A web-based platform that allows users to search for and book local service providers such as plumbers, electricians, cleaners, and technicians.

🚀 Features

- 🔍 Browse and search for local services
- 📅 Book appointments easily
- 👤 User authentication (Login / Register)
- 🧑‍🔧 Service provider dashboard
- 💬 Community chat system
- 🌙 Dark mode support
- 📱 Responsive design (mobile-friendly)

🧰 Technologies Used

- ASP.NET Core (MVC)
- Entity Framework Core
- SQL Server
- HTML / CSS / JavaScript
- Bootstrap

📂 Project Structure

- "Controllers/" → Handles requests
- "Models/" → Data models
- "Views/" → UI pages
- "Services/" → Business logic
- "Data/" → Database context
- "wwwroot/" → Static files (CSS, JS, images)

⚙️ Setup & Installation

1. Clone the repository:

git clone https://github.com/ahmed-mahmoud-io/LocalServicesBooking.git

2. Open the project in Visual Studio

3. Update "appsettings.json" with your database connection

4. Run migrations:

Update-Database

5. Run the project 🎉

🔐 Environment Variables

Make sure to configure:

- Database connection string
- Email settings (if used)
